# Fix Visual Studio 2019

* ##### Start Window
  * `Tools` > `Options` > `Environment` > `Startup`
  * On startup, open: > select `Empty environment`

* #### New Project Dialog
  * Install an extension called Classic "New Project" Dialog
  * After installing it is necessary to enable it in Options
    * `Tools` > `Options` > `Classic Project Dialog` > `General`
    * tick `Use Classic New Project Dialog`

* ##### Install an extension called "VSColorOutput"
